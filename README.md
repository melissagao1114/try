# Jeremy Quest

Mobile-first daily learning tracker built with React + TypeScript + Tailwind.

## Quick Start

```bash
npm install
npm run dev
```

If your environment enforces a private npm registry, override registry at runtime:

```bash
npm install --registry=<your_registry_url>
```

## Scripts

- `npm run dev` — start local dev server
- `npm run build` — type-check and production build
- `npm run preview` — preview production build
